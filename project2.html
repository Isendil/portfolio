Objective

In this project, my aim was to build an app from the ground up, to develop an entire back and front end application! I drew inspiration from both Netflix and IDMB, and aimed to create both an interactive and informative movie database. To get a feel for the project, you can take a look at my documentation below!

Check out the documentation here!

Description

In this project, I used a host of new technologies to develop a robust single page application. I started by building a backend using mongoDB and mongoDB atlas to create and host a small API of films I wanted to feature in the project. I then built a react application, to create an interactive front-end application using various technologies including redux to organise and manage the entire application's state.
