Objective

In this project, my aim was to use an external (meetup) API to engineer a real time event application using test driven development.
-AWS
-Jest, Enzyme, Cucumber - test driven development
-PWA
-Data Visualisation - recharts

Check out the app here!

Description

In this project, I used a host of new technologies to develop progressive web application. Initially I used AWS to create an authorization server to access Meetup - the external API which provided my app with a wealth of event data to play with! I then used Jest, Enzyme, Cucumber to produce the first few features of my project using a test driven development framework. To take the app to the next level I turned it into a progressive web application, allowing users the ability to use the app offline, and enabling users to download the app and generally allowing users to access latest versions without having the hassle of updating and re-installing the app. To top off the project, I incorporated Data visualisation using recharts, so users are presented with both a scatter chart of events in the local area over the week, and a few events display pie charts showing the number of available spaces for that particular event.
