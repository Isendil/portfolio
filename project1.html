The Rickopedia

Try it out here!

Objective

A quirky twist of the well known Pokedex and Wikipedia! This project aims to give users a fun and interactive encyclopedia of all the characters from the Rick and Morty series, using an external API (see link below for more information).

Check out the API here!

Description

This project was my first responsive application, tested thoroughly for multi-platform capabilities. It referenced a free, opensource API for all its data purposes (https://rickandmortyapi.com/) and was created as an exploration into the JQuery and Bootstrap libraries. It has a number of quirky features, my favourite being the infinite scroll feature that I implemented to load the data set. This feature means that every time a user reaches the bottom of the current page, the bottom of the div is used an event to be listened to, at which stage the next page of data is loaded and displayed! It was a necessary and fun feature to implement, as I found loading all of the data at once created a BIG lag, and didn't feel very user friendly!
