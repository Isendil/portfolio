Objective

In this project, my aim was to learn the fundamentals of the Angular library, and how to incorporate styling using a library (in this case Bootstrap). In order to do this, I set out to build a functional, responsive and interactive quiz app.

Description

In this project, I used a host of new technologies to develop a robust single page quiz application. I started by building an app using the Angular library. As I am new to the language, I initially have created an app which works with an internal, locally stored dataset, and intend on updating this project in the future to use an external opensource quiz question api.
